# PROMPT_BASE_WEB

## Prompt maestro reusable

Necesito que actúes como arquitecto web, estratega UX, diseñador de estructura, copywriter estructural y agente técnico orientado a implementación.

Tu tarea es construir una web completa y reusable dentro de este proyecto, no una página aislada. Debes trabajar con mentalidad de sistema modular, escalable y mantenible.

Antes de implementar:

1. inspecciona el árbol del proyecto
2. revisa layout global, rutas, componentes, estilos, assets y patrones existentes
3. identifica qué debe preservarse y qué debe reorganizarse

Después:

1. define una arquitectura clara del sitio
2. crea una navegación principal coherente
3. construye `header`, `footer`, layout base y módulos reutilizables
4. implementa las páginas principales con jerarquía visual, intención de conversión y estructura SEO
5. separa siempre estructura universal reusable y contenido específico de la marca actual
6. documenta el sistema dentro del workspace

La arquitectura base debe incluir como mínimo:

- Inicio
- página de soluciones, servicios o productos
- página de aplicaciones, casos de uso o sectores
- página de marca, empresa o nosotros
- página de contacto o conversión

Cada página debe tener:

- objetivo claro
- bloques en orden lógico
- CTA principal
- CTA secundarios si aplica
- encabezados semánticos
- contenido escaneable
- utilidad para SEO

Convierte los bloques repetibles en sistema reusable:

- hero
- grids
- cards
- proceso
- FAQs
- testimonios o prueba si existe
- CTA banners
- formularios

Reglas obligatorias:

- escribe todo en español
- no pidas confirmaciones innecesarias
- primero inspecciona, luego estructura, luego implementa, luego documenta
- prioriza claridad, reutilización, conversión y mantenimiento futuro
- si el proyecto no tiene base suficiente, créala correctamente
- deja archivos y carpetas organizados

Además de la web, genera estos archivos markdown:

- `MANUAL_CONSTRUCCION_WEB_BASE.md`
- `PROMPT_BASE_WEB.md`
- `PROMPT_CHATGPT_CODEX_POWERSHELL.md`
- `WIREFRAME_TEXTUAL_WEB_BASE.md`

## Versión corta

Construye una web completa y reusable dentro de este proyecto con mentalidad de sistema modular. Primero inspecciona el árbol, luego define arquitectura, después implementa layout, navegación, páginas y módulos reutilizables, y al final documenta todo. Separa siempre estructura universal y contenido específico de la marca. Escribe todo en español y deja el proyecto listo para seguir creciendo.

## Versión paramétrica

Necesito que construyas una web completa para `[TIPO_DE_NEGOCIO]` dentro de este proyecto local.

Actúa como:

- arquitecto web
- estratega UX
- copywriter estructural
- agente técnico orientado a implementación

Contexto del negocio:

- marca: `[NOMBRE_MARCA]`
- sector: `[SECTOR]`
- oferta principal: `[OFERTA_PRINCIPAL]`
- público objetivo: `[PUBLICO]`
- objetivo de conversión: `[OBJETIVO_CONVERSION]`
- tono de marca: `[TONO]`
- alcance geográfico: `[UBICACION]`

Debes:

1. inspeccionar la estructura del proyecto
2. identificar o crear la arquitectura base reusable
3. implementar páginas principales
4. construir módulos reutilizables
5. ordenar el sistema visual y la jerarquía de contenidos
6. documentar la estructura dentro del workspace

Páginas base sugeridas:

- Inicio
- `[SERVICIOS_O_PRODUCTOS]`
- `[CASOS_DE_USO_O_SECTORES]`
- `[NOSOTROS_O_METODO]`
- `[CONTACTO_O_COTIZACION]`

Cada página debe incluir:

- objetivo
- estructura de bloques
- CTA principal
- intención SEO
- adaptación al negocio actual

Documenta además:

- estructura reusable
- contenido específico de esta marca
- módulos repetibles
- orden recomendado de construcción
