Este directorio concentra activos reutilizables de marca y grafica para la web.

Estructura sugerida:

- `source-extractions/`: extracciones directas desde PDFs u otros documentos fuente
- `logo/`: versiones curadas del logo para header, footer, favicon y piezas de UI
- `palette/`: referencias de color, muestras y tokens visuales
- `patterns/`: texturas, fondos y elementos decorativos
- `photos/`: imagenes de apoyo optimizadas para web

Proceso base:

1. Extraer assets crudos desde el PDF o archivo fuente.
2. Identificar el logo maestro y mover una copia curada a `logo/`.
3. Documentar nombre, origen y uso recomendado.
